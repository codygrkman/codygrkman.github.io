    Instantory

    .NET 7 Blazor SQL (EF Core) Azure Identity Framework Core Bootstrap Font Awesome

    Instantory is a .NET 7 Blazor server application. This SPA was written for a small business client who was spending hundreds of man hours per year to keep their inventory accurate. Their old way of doing things was to print off a list of every item in their inventory (over 15,000 items!) and count them each by hand to ensure their counts were accurate. In 2021, I proposed a new solution that would allow them to simply scan the barcode of each item from their mobile phone and accurately update their inventory counts.

    This project required the use of .NET 7 Blazor server for the front-end, as well as a services pattern architecture and domain-driven design for the back-end and controller layers. I used a Microsoft stack the whole way through, by employing the use of Azure web services for hosting, .NET 7, Blazor, SQL (EF Core), Identity Framework Core for authorization, and OData for json interpretation of both inbound and outbound inventory records.

    Business Impact: A 400% reduction in the man hours required to complete yearly inventory.
